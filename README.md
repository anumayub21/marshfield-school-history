Marshfield School History Website

This project is a multi-page historical website created using:
HTML
CSS
PHP
MySQL

Features:
• Staff database search
• Historical source database
• PDF document access
• Multi-page navigation

To run the project:
1. Install XAMPP
2. Import marshfield_app.sql into phpMyAdmin
3. Place project in htdocs
4. Open http://localhost/enterprise_pro
